# 中文
这是一个植物大战僵尸融合版的小插件，用于显示植物/僵尸的属性
植物/僵尸的属性将随血量一起显示

**按键帮助**：

P键：是否显示植物属性
LeftShift + C/E: 切换中英文

**Release**

在项目的releases文件夹有构建好的dll
